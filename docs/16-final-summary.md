## 15. 最后一句话

GitHub 的核心并不复杂：

```text
找项目：搜索 + Star + 阅读 README
建项目：New repository + README + LICENSE + .gitignore
传代码：git add + git commit + git push
协作：Branch + Pull Request + Issue
```

只要按这个流程练几遍，你就能从 GitHub 小白变成能独立使用 GitHub 的开发者。
