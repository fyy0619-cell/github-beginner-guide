## 12. 推荐新手练习路线

### 第 1 天：会看

- 注册 GitHub。
- 搜索 5 个感兴趣项目。
- 学会看 README。
- 学会 Star 项目。

### 第 2 天：会建

- 新建自己的第一个仓库。
- 写 README。
- 添加 LICENSE。
- 添加 `.gitignore`。

### 第 3 天：会传

- 安装 Git。
- 配置用户名和邮箱。
- 本地 `git init`。
- `git add`、`git commit`、`git push`。

### 第 4 天：会改

- 修改 README。
- 再次提交。
- 查看提交历史。
- 学会回看 diff。

### 第 5 天：会协作

- Fork 一个仓库。
- Clone 到本地。
- 新建分支。
- 修改后提交 PR。
