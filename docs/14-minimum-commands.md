## 13. 建议你记住的最小命令集

如果你只想先记最常用的，记这些就够了：

```bash
git init
git status
git add .
git commit -m "说明"
git branch -M main
git remote add origin 仓库地址
git push -u origin main
git clone 仓库地址
git pull
git push
git switch -c 分支名
git switch main
```
